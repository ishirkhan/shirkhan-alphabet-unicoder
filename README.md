# 概述

JavaScript 允许采用 \uxxxx 形式表示一个字符，其中“xxxx” 表示字符的码点(code point)。
但是，这种表示法只限于 \u0000 —— \uFFFF 之间的字符。超出这个范围的字符，必须用两个双字节的形式表达。
